# Vrinda Store Sales Analysis (Excel Dashboard)

## Project Overview
I created an interactive Excel dashboard to analyze and visualize sales and order trends for Vrinda Store.  
This project helped me improve my skills in data cleaning, data processing, data analysis, and data visualization.

---

## Steps Followed in This Project

- Data Cleaning  
- Data Processing  
- Data Analysis  
- Data Visualization  
- Report Creation  
- Generating Insights and Business Recommendations  

---

## Key Insights from the Dashboard

- Compared **Sales vs Orders** in a single chart  
- Identified the **highest sales month of 2022**  
- **Women contributed 64%** of total purchases  
- **92% of orders were delivered successfully**  
- Top contributing states: **Maharashtra, Karnataka, Uttar Pradesh**  
- Analyzed **Age vs Gender buying trends**  
- Channels with maximum sales: **Amazon (35%) and Myntra (23%)**

---

## Final Conclusion to Improve Vrinda Store Sales

Vrinda Store should target **women customers aged 30–49 years** living in **Maharashtra, Karnataka, and Uttar Pradesh**.

The company can increase sales by focusing on:
- Targeted advertisements  
- Special offers and discount coupons  
- Promotions on major platforms such as **Amazon, Flipkart, and Myntra**
